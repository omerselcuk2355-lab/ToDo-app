# ToDo-app

HTML, CSS ve JavaScript kullanarak geliştirdiğim basit bir görev takip uygulaması.

## Özellikler

- Yeni görev ekleme
- Görev silme
- Görev düzenleme
- Todo / In Progress / Complete durum takibi
- Görev arama ve filtreleme
- Sayfa yenilenince veriler kaybolmuyor (localStorage)

## Kullanılan Teknolojiler

- HTML
- CSS (Glassmorphism tasarım)
- JavaScript (MVC yapısı)

## Nasıl Çalıştırılır

1. Repoyu klonlayın:
   git clone https://github.com/omerselcuk2355-lab/ToDo-app.git

2. Klasöre girin:
   cd ToDo-app

3. index.html dosyasını tarayıcıda açın.
   - Direkt çift tıklayarak açabilirsiniz
   - Ya da VS Code'da Live Server ile çalıştırabilirsiniz
